Bootstrap 4
Popover

Empat varian syntax popover dari dokumentasi W3Schools: basic, positioning, dismissible, dan hover trigger.

01 / 04

Basic Popover

Popover paling dasar. Perlu dua atribut: title untuk teks header, data-content untuk isi body. Tidak aktif otomatis - harus diinisialisasi via jQuery.

<a href="#" data-toggle="popover" title="Popover Header" data-content="Some content inside the popover"> Toggle popover </a> <script> $('[data-toggle="popover"]').popover(); </script>
Toggle popover

Klik tombol untuk toggle. Klik lagi di elemen yang sama untuk menutup.

02 / 04

Positioning

Atribut data-placement mengontrol dari sisi mana popover muncul. Empat nilai yang tersedia: top, bottom, left, right. Bootstrap otomatis flip ke sisi lain bila ruang tidak cukup.

<a href="#" data-toggle="popover" data-placement="top" title="Header" data-content="Content">Top</a> <a data-placement="bottom" ...>Bottom</a> <a data-placement="left" ...>Left</a> <a data-placement="right" ...>Right</a>
Top Bottom Left Right

Klik masing-masing tombol. Perhatikan arah popover berbeda-beda sesuai nilai data-placement.

03 / 04

Dismissible
Focus Trigger

data-trigger="focus" membuat popover tertutup saat user klik di luar elemen - tidak perlu klik tombol lagi. Harus pakai tag <a> dengan atribut tabindex="0" agar bekerja lintas browser.

<a href="#" tabindex="0" data-toggle="popover" data-trigger="focus" title="Dismissible popover" data-content="Klik di luar untuk menutup"> Click me </a>
Click me

Klik tombol, lalu klik area kosong di luar - popover otomatis tertutup.

04 / 04

Hover Trigger

data-trigger="hover" membuat popover muncul ketika mouse diarahkan ke elemen, dan hilang saat mouse meninggalkan elemen. Berguna sebagai pengganti tooltip biasa saat konten terlalu panjang.

<a href="#" data-toggle="popover" data-trigger="hover" title="Header" data-content="Some content"> Hover over me </a>
Hover over me

Arahkan mouse ke tombol - tidak perlu klik sama sekali.

Studi Kasus

Contoh Penggunaan dalam Bisnis

Popover bukan sekadar tooltip. Dalam konteks produk nyata, popover dipakai untuk menyampaikan informasi kontekstual tanpa memaksa user berpindah halaman.

Kasus 1 - E-Commerce

Informasi stok dan garansi produk

Laptop Ultrabook X1

Rp 14.500.000

<!-- hover: info stok realtime --> <button data-trigger="hover" title="Status Stok" data-content="Tersisa 3 unit..."> <!-- click: detail garansi --> <button data-toggle="popover" title="Garansi Resmi" data-content="Garansi 2 tahun...">

Kasus 3 - Form Registrasi

Panduan aturan input tanpa mengganggu layout

<!-- focus trigger: panduan muncul saat user aktif di field --> <a tabindex="0" data-trigger="focus" data-placement="right" title="Syarat Password" data-content="Min. 8 karakter..."> Lihat syarat </a>
